# Demo CRM

This repository is the planning and implementation home for a project-based B2B CRM.

## What This CRM Is For

The target business sells end-to-end data warehouse construction services, including design, materials, engineering, labor, and project delivery.

This is not a generic retail or SaaS workflow. The CRM is intended to support a realistic B2B sales-to-delivery process from lead capture through contract signing and project handoff.

## Current Stage

The project is currently in the repository foundation phase. It contains planning documents, repo rules, and a minimal implementation-ready structure only.

## Repository Foundation

The repo now includes the approved planning and foundation layer:

- documentation for business scope, MVP scope, architecture, phases, conventions, and the initial data model,
- repository-level guidance for Codex collaboration,
- a minimal future app directory layout without any initialized framework code,
- placeholder root configuration files for environment variables and ignore rules.

No dependencies have been installed, no framework has been initialized, and no business modules have been scaffolded yet.
